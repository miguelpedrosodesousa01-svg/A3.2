🚀 INÍCIO RÁPIDO - 5 MINUTOS
═════════════════════════════════════════════════════════════════════════════

PASSO 1: Verificar Java
─────────────────────
  Abra Command Prompt e execute:
    java -version

  Resultado esperado: Java 11 ou superior


PASSO 2: Obter Driver MySQL
──────────────────────────
  
  Opção A (Automático):
    1. Clique direito em: download_mysql_driver.ps1
    2. Selecione: Run with PowerShell
  
  Opção B (Manual):
    1. Acesse: https://dev.mysql.com/downloads/connector/j/
    2. Versão: 8.0.33
    3. Salve em: lib/mysql-connector-java-8.0.33.jar


PASSO 3: Criar Banco de Dados
──────────────────────────────
  
  Abra Command Prompt e execute:
    mysql -u root -p < "C:\Users\migue\Desktop\a3 final\database\schema.sql"
  
  Ou manualmente:
    1. mysql -u root -p
    2. source C:\Users\migue\Desktop\a3 final\database\schema.sql;


PASSO 4: Compilar
─────────────────
  
  Duplo clique em: compile_simple.bat


PASSO 5: Executar
─────────────────
  
  Abra Command Prompt e execute:
    cd "C:\Users\migue\Desktop\a3 final"
    java -cp "bin;lib\mysql-connector-java-8.0.33.jar" Main

❌ TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Problema: "java: command not found"
  ➜ Solução: Instale Java JDK 11+
    https://www.oracle.com/java/technologies/downloads/

Problema: "Unknown database 'gestao_projetos'"
  ➜ Solução: Execute database/schema.sql no MySQL

Problema: "Access denied for user 'root'"
  ➜ Solução: Corrija senha em ConexaoDB.java

Problema: "no suitable driver found"
  ➜ Solução: Certifique-se de lib/mysql-connector-java-8.0.33.jar existir

Problema: "java.lang.ClassNotFoundException: com.mysql.cj.jdbc.Driver"
  ➜ Solução: Inclua -cp lib/mysql-connector-java-8.0.33.jar ao executar
